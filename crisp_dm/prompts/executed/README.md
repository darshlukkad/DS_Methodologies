# Executed Prompts Directory

This directory contains timestamped logs of all prompts and critiques executed during the CRISP-DM workflow.

## Format

Each file follows the naming convention:
```
YYYYMMDD_HHMMSS_<phase_name>.md
```

Example: `20251102_143052_business_understanding.md`

## Contents

Each executed prompt file includes:

1. **Phase Name**: Which CRISP-DM phase
2. **Timestamp**: When it was executed
3. **Original Prompt**: The exact prompt given to the assistant/LLM
4. **Response**: The complete response received
5. **Critique**: Expert review from Dr. Cassie Kozyrkov
6. **Actions Taken**: Specific changes made based on critique

## Purpose

- **Reproducibility**: Track decision-making process
- **Learning**: Review what worked and what didn't
- **Auditing**: Understand model development history
- **Knowledge Transfer**: Help others learn from the process
